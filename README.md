# guip
Ferramenta para obter seu IP e rastrear qualquer outro!
<h3> Instalação: </h3>

* apt-get install python3

* apt-get install git

* pip install requests

* git clone https://github.com/NerdAugustoSilva/guip

* cd guip

* python3 guip.py
